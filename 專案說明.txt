📊 專案概述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
這是一個全功能的美國股市分析系統，專注於三大指數的分析與視覺化。

✨ 核心功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 三大指數監控
   • NASDAQ (^IXIC) - 納斯達克綜合指數
   • 道瓊工業指數 (^DJI)
   • S&P 500 (^GSPC)

✅ K線圖視覺化
   • 紅色 K 線：收盤上漲
   • 綠色 K 線：收盤下跌
   • 顯示開盤、收盤、最高、最低價、成交量

✅ 相關性分析
   • 計算成分股與指數的皮爾森相關係數
   • 自動評級：高度、中度、低度、弱相關
   • 彩色標籤區分

✅ 動態端口檢測
   • 自動掃描 3000-3100 範圍
   • 避免端口衝突

🎨 使用指南
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 選擇指數
   點擊頂部標籤切換: NASDAQ | 道瓊工業指數 | S&P 500

2. 查看 K 線圖
   🔴 紅色柱 = 上漲  |  🟢 綠色柱 = 下跌
   滑鼠懸停查看詳細數據

3. 相關性分析
   查看成分股與指數的相關係數
   • 🔴 高度相關 (|r| ≥ 0.8)
   • 🟡 中度相關 (|r| ≥ 0.5)
   • 🔵 低度相關 (|r| ≥ 0.3)
   • ⚪ 弱相關   (|r| < 0.3)

4. 統計數據
   • 當前價格
   • 漲跌幅百分比
   • 成交量

🐛 故障排除
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 環境檢測
   python3 test_setup.py

2. 端口被占用
   後端: 修改 app.py 中的端口
   前端: 自動檢測會找到可用端口

3. 無法獲取數據
   • 檢查網絡連接
   • Yahoo Finance 可能有頻率限制
   • 確認股票代碼正確

4. 依賴問題
   後端: cd backend && source venv/bin/activate && pip install -r requirements.txt
   前端: cd frontend && rm -rf node_modules && npm install
